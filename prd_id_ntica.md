# PRD: IDÉNTICA - App de Asesoría de Imagen con IA

## Objetivo
Aplicación de asesoría de imagen personalizada basada en análisis visual por IA y datos del usuario.

## Estilo Visual
- **Nombre:** IDÉNTICA
- **Estética:** Moderna, colorida, llamativa, vibrante, limpia y visual.
- **Tipografía:** Sans serif.
- **Paleta:** Cálida y armónica.
- **Tono:** Femenino, elegante y profesional.

## Flujo de Pantallas
1. **Bienvenida:** Marca, eslogan "Descubre tu mejor versión" y botón "Comenzar". Ilustración femenina.
2. **Carga de Imagen:** Instrucciones para subir foto, botones de cámara/galería.
3. **Análisis IA:** Simulación visual de escaneo con barra de progreso y textos dinámicos (proporciones, piel, colorimetría).
4. **Formulario:** Datos de estatura, peso, tallas (camisa/pantalón) y ajuste (Ajustado, Regular, Oversize).
5. **Resultados:** 
   - Tipo de cuerpo (siluetas).
   - Datos ingresados (exactos).
   - Colorimetría (piel, subtono, ojos, cabello).
   - Estación de color y paleta visual.
   - Características visuales y recomendaciones de prendas.
6. **Pantalla Final:** Resumen con opciones de guardar, ver más o volver a analizar.
